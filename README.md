# AI Vision Classifier 🖼️

Bu proje, bir model kullanarak görüntü sınıflandırması yapan modern bir Streamlit uygulamasıdır. MobileNetV2 tabanlı bir CNN modeli kullanılarak eğitilmiştir.

## Özellikler
- 🖼️ **Görüntü Yükleme**: Kullanıcılar kendi görsellerini yükleyebilir.
- 🚀 **Anlık Tahmin**: Yapay zeka modeli anlık analiz yapar.
- 📊 **Detaylı Analiz**: İlk 3 tahmin ve olasılıkları grafiksel olarak gösterilir.
- 🎨 **Modern Arayüz**: Şık ve kullanıcı dostu tasarım.

## Kurulum ve Çalıştırma

### Yerel Çalıştırma için:
1. Gerekli kütüphaneleri yükleyin:
   ```bash
   pip install -r requirements.txt
   ```
2. Uygulamayı başlatın:
   ```bash
   streamlit run app.py
   ```

## Teknolojiler
- **Frontend/Uygulama**: Streamlit
- **Derin Öğrenme**: TensorFlow / Keras
- **Görüntü İşleme**: PIL (Pillow), Numpy
